# O que é o projeto HumanResources?

> É um sistema simples para cadastro de funcionários baseado nas tecnologias atuais.

** Tecnologias **

- ASP.NET Core 3.1
- Entity Framework Core 3.1
- NET Core Native DI
- AutoMapper
- FluentValidato
- Angular
